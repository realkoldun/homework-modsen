async и await -- помечаются асинхронные операции
async function -- возвращает Promise
Цепочки не поддерживаются напрямую (можно несколько await)

Виды ошибок:
    Ошибка синтаксиса -- код не соотв js
    Ошибка времени выполнения -- ошибки возник во время выполнения кода
    ReferenceError -- использование переменной до ее объявления или вне области видимости переменной
    TypeError -- несовместимость типов

DOM (Document Object Module) -- представление данной страницы в браузере в виде объекта
document в js
состоит из:
    Документы, элементы, аттрибуты, текстовые узлы
Дейстивия с DOM:

BOM (Browser Object Module) -- интрефейс для взаимодействия с браузером
Объекты DOM:
    window -- доступ ко всему контенту
    document -- объект связанный с DOM
    navigator -- инфа о браузере и устройстве
    location -- представляет url и методы его представления
    history -- управление историей браузера back() forward()
    screen -- инфа о экране
