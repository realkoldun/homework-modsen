Асинхронное программирование -- способномть выполнять операции не блокируя выполнения остальной части программы
пример
    сетевые запросы
    обработка событий
    обработка нескольких задач
    таймеры

Синхронный код -- одно операция -> другая операция

Ajax -- набор технологий, для создания асинхронных запросов серверов
без необходимости перезагружать веб страницу

setTimeout -- выполнение кода через задынный интервал
setInterval -- запускает ф-цию через интервал многократно
const interval = setInterval(() => {
    //код
    clearInterval(intervalId) // начинается сначала
}, 1000); //возвращает число в intervalId;

EventLoop является решение проблемы блокировки выполнения для длительных задач
EventLoop -- Бесконечный цикл выполнения команд имеющих приоритет
Основные составляющие EventLoop:
    Call Stack (стек вызовов) -- структура данных управляющая выполнением ф-ций (все синхронные ф-ции)
    Heap (куча) -- обьекты, хранение данных
    Task Queue (Очередь задач)
    отдельно Web APIs(Браузерные API)
